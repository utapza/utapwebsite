## Implementation Notes

**Platform:** Resend — the `send-cheat-sheet.ts` Netlify function sends Email 1 immediately. Emails 2–5 need a drip sequence configured in Resend's broadcast/audience feature, or a simple cron-triggered function using stored email + timestamp from the initial submission.

**From address:** `uTap <hello@utaptech.co.za>` — already configured in the function.

**Unsubscribe:** Currently handled manually (reply-based). When volume grows, add a Resend unsubscribe link in the footer of each email.

**Personalisation:** These emails do not use first name — we don't collect it. If we add a name field to the cheat sheet form, add `Hey [Name],` to Email 1 and propagate through.

**A/B test candidates:**
- Email 3 subject: "What a R100 campus meal actually costs you" vs "You're paying R75 in fees you don't have to pay"
- Email 4 subject: "500 students get this badge" vs "The Founding Student badge closes this week"
